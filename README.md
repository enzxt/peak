# peak

`peak` is a Solana project built around one simple idea: holding should feel alive.

The project starts with a single-page app that makes holder activity visible and rewards transparent. Instead of relying on vague promises, `peak` is designed around public rules, a visible reward reserve, and gradual rollout.

## Core principles

- Rewards must be sustainable.
- Rules should be published before they are used.
- The community should benefit as a whole.
- Competition should add energy without turning holders against each other.
- The first version should stay intentionally small.

## What we are building first

Version 1 focuses on a narrow system we can explain clearly and operate reliably:

- A live holder leaderboard
- A transparent reward reserve policy
- Scheduled snapshot-based rewards
- Public documentation for reward logic
- A single-page project site

## Public build philosophy

This repository is intended to be public. The goal is to show that `peak` is being built deliberately:

- design first
- math before marketing
- small releases
- transparent iteration

As the project evolves, this repo should include both the app and the underlying reward/tooling logic.
